# 🌸 Meu Portfólio em Flask

Bem-vindo(a) ao meu portfólio! 🚀  
Este projeto foi desenvolvido em **Python (Flask)**, com design moderno, está hospedado gratuitamente no **Vercel**.  

## ✨ Tecnologias Utilizadas
- Python 3  
- Flask  
- HTML5 + CSS3  
- Font Awesome (ícones)  
- Vercel (deploy)
